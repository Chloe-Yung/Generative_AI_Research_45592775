# Generative AI Research Project

## Project Overview
This repository contains research materials for the study:
"Using Generative AI Tools – Boon or Bane".

## Folder Structure
- Literature_Review: academic articles and references
- Quantitative_Data: survey data and analysis files
- Qualitative_Data: interview transcripts and insights
- Drafts_and_Reports: research drafts and final reports
- Visualisations: charts and figures

## Version Control
GitHub is used to track changes and manage collaboration.
Each update is committed with meaningful messages.

## Collaboration
Team members contribute using branches and pull requests.
